# Development and Validation of an Explainable Machine Learning Model for Predicting Early Clinical Deterioration in Guillain–Barré Syndrome Using Pretreatment Clinical Data

## Overview

This repository documents the research and development process of an undergraduate research project titled **"Development and Validation of an Explainable Machine Learning Model for Predicting Early Clinical Deterioration in Guillain–Barré Syndrome Using Pretreatment Clinical Data."** The project aims to develop an explainable machine learning-based clinical decision support system that predicts the risk of early clinical deterioration in newly diagnosed Guillain–Barré Syndrome (GBS) patients using only pretreatment clinical information available at hospital admission.

The repository serves as a centralized workspace for organizing topic exploration, literature review, research gap analysis, dataset planning, modeling methodology, and project documentation. It also tracks the progress of the research from initial idea development to model implementation and evaluation.
